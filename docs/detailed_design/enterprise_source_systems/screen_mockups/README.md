# Screen Mockups

## Purpose

This folder contains app-style and screen-level mockups for Integrity Sentinel detailed design workflows.

Each workflow has its own subdirectory so new areas can be added without mixing screen sets.

## Workflow Sets

| Workflow | Directory | Description |
|---|---|---|
| Admin Configuration Workflow | `admin_configuration_workflow/` | Tenant Admin flow for selecting a connector from the catalog and configuring governed ingestion. |

## Planned Workflow Sets

Open `admin_configuration_workflow/index.html` in a browser to review the interactive Connector Catalog and connector configuration prototype.
